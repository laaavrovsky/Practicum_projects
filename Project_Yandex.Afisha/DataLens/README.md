# Дашборд в DataLens: аналитика сервиса Яндекс.Афиша

## Описание:

В рамках проекта разработан аналитический дашборд в Yandex DataLens для мониторинга ключевых продуктовых метрик сервиса Яндекс.Афиша.

Дашборд позволяет отслеживать динамику выручки, анализировать поведение пользователей и исследовать структуру продаж в разрезе устройств, типов мероприятий и регионов.

## Цель:

Создать инструмент для продуктовой команды, который позволяет:

* отслеживать ключевые бизнес-метрики
* анализировать популярность мероприятий
* исследовать структуру выручки
* находить наиболее прибыльные сегменты

## Что реализовано:

### Ключевые метрики (индикаторы):

На дашборде выведены агрегированные показатели за выбранный период:

* общая выручка
* количество заказов
* средний чек
* количество уникальных пользователей

### Динамика показателей:

Реализованы линейные графики с недельной агрегацией:

* выручка
* количество заказов
* средний чек

Позволяют отслеживать изменения ключевых метрик во времени и выявлять тренды.

### Структура выручки:

Добавлен анализ структуры выручки:

* по типам устройств (mobile / desktop)
* по типам мероприятий

Позволяет понять, какие сегменты формируют основную выручку.

### Топ-сегменты:

Реализованы таблицы с ключевыми сегментами:

* регионы
* события
* площадки

Для каждого сегмента рассчитаны:

* общая выручка
* количество заказов
* среднее количество билетов в заказе
* средняя выручка с одного билета

### Фильтрация:

Добавлены фильтры для гибкого анализа данных:

* по валюте заказа (по умолчанию - RUB)
* по дате (по умолчанию: июнь - октябрь 2024)
* по типу событий
* по региону

Фильтры позволяют детализировать анализ и исследовать данные в различных разрезах.

## Особенности:

* Дашборд построен на продуктовых метриках (выручка, заказы, средний чек)
* Реализована связка с SQL-расчётами
* Добавлена гибкая система фильтрации
* Дашборд ориентирован на принятие решений, а не просто визуализацию

## Результат:

* Разработан инструмент для мониторинга продуктовых метрик
* Обеспечена возможность анализа структуры выручки
* Реализован анализ ключевых сегментов
* Улучшена прозрачность данных для продуктовой команды
