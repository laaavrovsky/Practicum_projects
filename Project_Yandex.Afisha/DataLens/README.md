## На основе запросов в SQL был создан интерактивный дашборд

Ссылка на дашборд в DataLens:
https://datalens.ru/psu2gcsz857e9

Дашборд разработан для продуктовой команды. Он должен помочь:
* отслеживать динамику ключевых бизнес-показателей;
* анализировать популярность мероприятий;
* исследовать структуру выручки в разрезе категорий событий и устройств.

**Основная цель** - дать продуктовой команде быстрый доступ к ключевым метрикам и упростить принятие решений на основе данных.
