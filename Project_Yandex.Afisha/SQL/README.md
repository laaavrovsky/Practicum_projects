# SQL-анализ данных сервиса Яндекс.Афиша

## Описание:

В рамках проекта проведён анализ продуктовых данных сервиса Яндекс.Афиша с использованием SQL.

Цель - рассчитать ключевые метрики, изучить структуру выручки и пользовательского поведения, а также подготовить данные для построения дашборда и дальнейшего анализа.

Анализ выполнен на данных заказов билетов с учётом мультивалютности (рубли и тенге).

## Задачи:

* Рассчитать ключевые бизнес-метрики сервиса
* Проанализировать структуру выручки
* Изучить поведение пользователей по устройствам
* Выявить различия между типами мероприятий
* Построить динамику ключевых показателей
* Определить наиболее прибыльные регионы

## Что было сделано:

* Рассчитал агрегированные метрики за весь период:

  * общая выручка
  * количество заказов
  * средний чек
  * уникальные пользователи

* Учёл мультивалютность данных (RUB / KZT) и провёл анализ по валютам

* Проанализировал распределение выручки и заказов по типам устройств:

  * оценил долю выручки (revenue_share)
  * сравнил поведение пользователей mobile и desktop

* Исследовал структуру выручки по типам мероприятий:

  * определил наиболее прибыльные категории
  * оценил средний чек, количество билетов и выручку на билет
  * рассчитал долю выручки по сегментам

* Построил недельную динамику ключевых метрик:

  * выручка
  * количество заказов
  * уникальные пользователи
  * средний чек

* Выделил топ-7 регионов по выручке:

  * проанализировал объём продаж
  * оценил количество клиентов и билетов
  * рассчитал среднюю стоимость билета

## Структура:

* `general_values_of_key_indicators.sql` - расчёт ключевых метрик
* `revenue_distribution_by_device.sql` - анализ по устройствам
* `revenue_distribution_by_event.sql` - анализ по типам мероприятий
* `dynamics_of_value_changes.sql` - динамика показателей
* `top_segments.sql` - топ регионов по выручке

## Особенности:

* Учтена мультивалютная структура данных
* Использованы продуктовые метрики (выручка, MAU, средний чек)
* Проведена сегментация по устройствам и категориям событий
* Подготовлена база для построения дашборда и аналитики
