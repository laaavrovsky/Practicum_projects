## SQL-часть проекта

В рамках проекта были написаны SQL-запросы для решения нескольких задач: 
* вычисление общих значений ключевых показателей сервиса за весь период;
* вычисление распределения выручки и количества заказов по типу устройства для заказов в рублях;
* вычисление распределения выручки и количества заказов в зависимости от типа мероприятия для заказов в рублях;
* вычисление изменения выручки, количества заказов, уникальных клиентов и средней стоимости одного заказа в недельной динамике для заказов в рублях;
* выделение топ-7 регионов по значению общей выручки для заказов в рублях.

**В дальнейшем полученные результаты использовались при построении дашборда.**

Запросы из учебного SQL-тренажера внесены в папку `SQL` и оформлены отдельными файлами для удобства просмотра.
