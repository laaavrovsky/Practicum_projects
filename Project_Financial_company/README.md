# Анализ данных финансовой компании

## О проекте:

В проекте проведён анализ данных финансовой компании с целью изучить поведение клиентов, оценить ключевые бизнес-показатели и выявить факторы, влияющие на финансовые результаты.

Анализ основан на данных о клиентах, их активности и финансовых операциях. Проект имитирует задачи аналитика в финансовом секторе: от исследования данных до формирования выводов для бизнеса.

## Цели исследования:

* Проанализировать структуру клиентской базы
* Оценить ключевые финансовые показатели
* Выявить паттерны поведения клиентов
* Определить факторы, влияющие на доходность
* Сформулировать аналитические выводы и рекомендации

## Что было сделано:

* Провёл предобработку данных:

  * обработал пропуски и аномалии
  * проверил корректность типов данных
* Выполнил исследовательский анализ (EDA):

  * изучил распределения ключевых показателей
  * проанализировал поведение клиентов
  * выявил зависимости между признаками
* Сегментировал клиентов по различным признакам
* Проанализировал финансовые метрики и их динамику
* Выявил факторы, влияющие на ключевые показатели
* Сформулировал выводы и интерпретировал результаты с точки зрения бизнеса

## Основные направления анализа:

* Анализ клиентской активности
* Исследование финансовых показателей
* Сегментация клиентов
* Поиск закономерностей в данных

## Стек:

* Python
* `pandas` — обработка данных
* `numpy` — численные операции
* `matplotlib`, `seaborn` — визуализация данных
