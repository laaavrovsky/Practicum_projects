# SQL-анализ данных сервиса Яндекс Книги

## Описание

В данной части проекта проведён анализ пользовательской активности сервиса Яндекс Книги с использованием SQL.

Данные содержат информацию о чтении и прослушивании контента: пользователи, авторы, произведения, жанры, длительность сессий и платформы. Анализ выполнен на данных за период с 1 сентября по 11 декабря 2024 года.

Цель - исследовать поведение пользователей, оценить популярность контента и рассчитать ключевые продуктовые метрики.

## Задачи:

* Проанализировать пользовательскую активность и вовлечённость
* Выявить наиболее популярный контент (авторы и произведения)
* Оценить удержание пользователей после маркетинговой активности
* Рассчитать монетизационные метрики (LTV, выручка)

## Что было сделано:

* Рассчитал MAU авторов и выделил топ-3 авторов по активности пользователей
* Рассчитал MAU произведений, определил топ-3 произведений и их характеристики (жанры, авторы)
* Проанализировал ежедневный Retention Rate пользователей, активных в день рекламной кампании (2 декабря)
* Рассчитал средний LTV пользователей для Москвы и Санкт-Петербурга
* Оценил эффективность монетизации через метрику выручки на час потребления контента
* Построил агрегированные метрики: MAU, суммарное время потребления и выручка в динамике

## Структура:

* `MAU_Authors.sql` — расчёт MAU авторов
* `MAU_Content.sql` — расчёт MAU произведений
* `Retention_Rate.sql` — анализ удержания пользователей
* `LTV.sql` — расчёт LTV по городам
* `Average_Revenue.sql` — выручка на час потребления контента
