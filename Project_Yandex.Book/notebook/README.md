# Анализ данных и A/B-тестирование в сервисе Яндекс Книги

## Описание:

Во второй части проекта проведён исследовательский анализ данных и проверка продуктовых гипотез для сервиса Яндекс Книги.

Работа разделена на две части:

1. Проверка гипотезы о различиях в пользовательском поведении
2. Анализ результатов A/B-тестирования

Цель - выявить особенности пользовательской активности и оценить влияние продуктовых изменений на поведение пользователей.

## Цели исследования:

* Проверить гипотезу о различиях в активности пользователей из Москвы и Санкт-Петербурга
* Проанализировать поведение пользователей на основе данных об активности
* Оценить корректность проведения A/B-теста
* Проанализировать результаты эксперимента и сделать выводы

## Часть 1. Проверка гипотезы:

**Гипотеза:**
Пользователи из Санкт-Петербурга проводят в среднем больше времени в сервисе, чем пользователи из Москвы.

### Что было сделано:

* Подготовил данные:

  * проверил дубликаты пользователей
  * исключил пересечения между группами
  * проанализировал распределения и выбросы
* Сравнил размеры выборок и базовые статистики
* Провёл статистический тест (t-test для независимых выборок)
* Интерпретировал результаты на основе p-value

### Результат:

Сформулирован вывод о наличии / отсутствии статистически значимых различий между группами и предложены возможные объяснения наблюдаемого поведения пользователей.

## Часть 2. Анализ A/B-тестирования:

### Что было сделано:

* Загрузил и предобработал данные эксперимента
* Проверил корректность проведения теста:

  * пересечения пользователей между группами
  * баланс выборок
  * корректность распределения пользователей
* Проанализировал ключевые метрики в группах
* Оценил статистическую значимость различий
* Сделал вывод о результатах эксперимента

## Итог:

* Проверена продуктовая гипотеза о различиях в пользовательском поведении
* Проведён полный цикл анализа A/B-теста: от проверки данных до интерпретации результатов
* Сформированы выводы, которые могут быть использованы для принятия продуктовых решений
