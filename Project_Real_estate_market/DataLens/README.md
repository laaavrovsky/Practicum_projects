# Дашборд в DataLens: анализ рынка недвижимости

## Описание:

В рамках проекта доработан аналитический дашборд для агентства недвижимости, планирующего выход на рынок Санкт-Петербурга и Ленинградской области.

Цель - превратить существующий дашборд в инструмент для принятия бизнес-решений: добавить возможности сегментации, углубить анализ и улучшить визуализацию ключевых метрик.

## Задачи:

* Расширить возможности фильтрации данных
* Добавить новые продуктовые метрики
* Улучшить читаемость и интерпретацию данных
* Реализовать инструменты для анализа сезонности и скорости продаж
* Сделать дашборд удобным для принятия решений

## Что было сделано:

### Улучшение метрик:

* Добавил показатель **средней площади квартир**
* Рассчитал и визуализировал **среднее время продажи (days_exposition)**
* Обеспечил единый формат отображения метрик (включая area chart для сопоставимости)

### Расширение фильтрации:

* Добавил фильтр по **типу населённого пункта**
* Реализовал фильтр по **временным периодам** (день, неделя, месяц, год)

### Улучшение анализа по локациям:

* Заменил технические `city_id` на **названия населённых пунктов**
* Отсортировал данные по **стоимости квадратного метра**
* Добавил **цветовую градацию (heatmap)**:

  * дорогие объекты - красные
  * средние - жёлтые
  * доступные - зелёные

### Анализ скорости продаж:

* Построил чарт **топ-5 населённых пунктов с минимальным временем продажи**
* Добавил цветовую дифференциацию для визуального сравнения скорости продаж

### Улучшение визуализаций:

* Заменил круговые диаграммы на **столбчатые** для корректного отображения средних значений
* Добавил новые признаки:

  * наличие водоёма
  * наличие парка
* Проанализировал влияние этих факторов на стоимость недвижимости

### Анализ поведения пользователей:

* Определил **дни недели с максимальной и минимальной активностью публикаций**
* Реализовал цветовую подсветку:

  * самый активный день - зелёный
  * наименее активный - красный

### Анализ сезонности:

* Построил график динамики объявлений:

  * по месяцам
  * с разбивкой по годам
* Выявил сезонные паттерны и изменения активности на рынке

## Особенности:

* Дашборд доработан под реальные бизнес-требования
* Улучшена читаемость и интерпретация данных
* Добавлены инструменты для глубокой сегментации
* Реализован анализ, ориентированный на принятие решений
