# SQL-анализ рынка недвижимости (Санкт-Петербург и Ленинградская область)

## Описание:

В рамках проекта проведён анализ объявлений о продаже жилой недвижимости для агентства, планирующего выход на рынок Санкт-Петербурга и Ленинградской области.

Цель - выявить наиболее перспективные сегменты недвижимости и определить ключевые закономерности рынка: скорость продажи объектов и сезонность активности.

Анализ выполнен на данных сервиса Яндекс Недвижимость за 2015 - 2018 годы.

## Задачи:

* Оценить скорость продажи объектов недвижимости
* Выявить различия между рынком Санкт-Петербурга и Ленинградской области
* Проанализировать сезонные колебания активности продавцов и покупателей
* Определить характеристики объектов в разные периоды

## Что было сделано:

### 1. Анализ времени активности объявлений

* Рассчитал длительность публикации объявлений (в днях)
* Сегментировал объявления по категориям времени активности:

  * до 30 дней
  * 31 - 90 дней
  * 91 - 180 дней
  * более 180 дней
  * отдельная категория - активные объявления без даты снятия
* Сравнил категории по ключевым характеристикам:

  * средняя стоимость за м²
  * средняя площадь
  * количество комнат и балконов
* Провёл сравнительный анализ между Санкт-Петербургом и городами Ленинградской области

### 2. Анализ сезонности объявлений:

* Проанализировал динамику публикации и снятия объявлений по месяцам
* Выявил периоды с повышенной активностью на рынке
* Оценил сезонные изменения в характеристиках недвижимости
* Использовал только полные годы (2015 - 2018) для корректного анализа

## Особенности анализа:

* Учтена специфика рынка недвижимости (снятие объявления как прокси продажи)
* Проведена сегментация по времени экспозиции объектов
* Исключены неполные периоды для устранения искажений
* Анализ ориентирован на практическое применение в бизнесе

## Результат:

* Выявлены сегменты недвижимости с разной скоростью продажи
* Определены различия между рынками Санкт-Петербурга и области
* Найдены сезонные паттерны активности
* Подготовлена аналитическая база для разработки стратегии выхода на рынок
