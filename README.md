### Yandex.Afisha - продуктовая аналитика

**Цель:**
Проанализировать продуктовые метрики, понять изменения в поведении пользователей и выявить драйверы выручки.

**Масштаб:**
Данные заказов и пользователей за несколько месяцев (~сентябрь - декабрь 2024), десятки тысяч записей.

**Что сделал:**

* Рассчитал ключевые продуктовые метрики в SQL (выручка, заказы, пользователи, средний чек)
* Построил аналитический дашборд в DataLens
* Провёл EDA и проверку гипотез в Python
* Проанализировал популярность событий и поведение пользователей
* Сравнил mobile и desktop сегменты

**Стек:** SQL · Python · DataLens

### Yandex.Books - анализ поведения пользователей и A/B-тестирование

**Цель:**
Изучить пользовательскую активность и проверить продуктовые гипотезы.

**Масштаб:**
Данные о чтении и прослушивании контента за период с 1 сентября по 11 декабря 2024.

**Что сделал:**

* Рассчитал метрики в SQL (MAU, retention, LTV)
* Проанализировал поведение пользователей и потребление контента
* Проверил гипотезу (Москва vs Санкт-Петербург)
* Провёл полный анализ A/B-теста

**Стек:** SQL · Python
